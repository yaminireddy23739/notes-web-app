# Notes Web App

A simple and responsive Notes Web Application built using HTML, CSS, and JavaScript.

## Features
- Add Notes
- Delete Notes
- Search Notes
- Data stored using LocalStorage
- Clean and responsive UI

## Technologies Used
- HTML5
- CSS3
- JavaScript 
- LocalStorage API

##  What I Learned
- DOM Manipulation
- Event Handling
- Local Storage usage
- UI Styling with CSS
- Building complete frontend projects from scratch
